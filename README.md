## Model Explanation

### What is DeepTaxa?

DeepTaxa classifies 16S rRNA gene sequences into the seven standard taxonomic ranks (Domain → Phylum → Class → Order → Family → Genus → Species) in a single forward pass. It was trained on the **Greengenes2 2024.09** release (277,336 training / 69,335 test full-length 16S sequences).

### Architecture: Hybrid CNN-BERT

```
Input 16S rRNA sequence (tokenised via DNABERT-2 BPE, max_length=512)
        │
        ▼
  [Embedding]  embed_dim=896
        │
   ┌────┴─────────────────────┐
   │ CNN Branch               │ BERT Branch
   │ 3 × Conv1d stacks        │ 4-layer transformer
   │ kernels=[3,5,7]          │ hidden=896, heads=7
   │ filters=256/kernel       │ intermediate=3584
   │ ReLU + Dropout(0.1)      │
   │ Global max pool          │ CLS token
   │ Linear → 896             │
   └──────────────┬───────────┘
                  │ Fusion: α·CNN + β·BERT + BERT (residual)
                  │ Dropout(0.1)
                  │
        ┌─────────┼─────────┐ ... (7 heads)
        ▼         ▼         ▼
    [Domain]  [Phylum]  [Species]
    Linear    Linear    Linear
    → n_d     → n_p     → n_s
```

#### Key design choices

| Component | Detail |
|-----------|--------|
| **Tokenizer** | DNABERT-2 BPE (`zhihan1996/DNABERT-2-117M`), vocab_size=4096 |
| **Embedding** | Learned, embed_dim=896, tied to BERT hidden size |
| **CNN branch** | 3 Conv1d stacks × 3 kernel sizes (3, 5, 7) → 256 filters each; global max pool |
| **BERT branch** | 4 transformer layers, 7 attention heads, intermediate=3584 |
| **Fusion** | α·CNN_proj + β·BERT_CLS + BERT_CLS (residual); α and β are learnable scalars |
| **Classifiers** | 7 independent linear heads (one per taxonomic rank) |
| **Total params** | ~76.4M |

#### Why CNN + BERT?

- **CNN** excels at detecting short, conserved sequence motifs (primer-binding sites, hypervariable regions) efficiently.
- **BERT** captures long-range dependencies and global sequence context via self-attention.
- The **learnable fusion** lets the model discover the optimal balance between local and global features per task.

### Performance

| Rank | Accuracy |
|------|----------|
| Domain | 99.97% |
| Phylum | 99.52% |
| Class | 98.82% |
| Order | 97.73% |
| Family | 96.74% |
| Genus | 95.01% |
| **Species** | **92.96%** |

### Quick-start (CLI)

```bash
pip install deeptaxa
deeptaxa predict \
    --model-path deeptaxa-full-length-v1.pt \
    --input my_sequences.fna \
    --output-dir predictions/
```

### Training

```bash
deeptaxa train \
    --train-data gg2_train.fna \
    --test-data  gg2_test.fna \
    --output-dir checkpoints/ \
    --epochs 30 --batch-size 32
```
